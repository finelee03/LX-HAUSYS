# AUD Website

LX Hausys AUD(Architecture & Urban Design) 웹사이트 프로젝트

## 프로젝트 소개

LX Hausys의 건축 및 도시 디자인 솔루션을 소개하는 웹사이트입니다.

## 기술 스택

- HTML5
- CSS3
- JavaScript

## 시작하기

```bash
# 레포지토리 클론
git clone https://github.com/finelee03/aud-website.git

# 프로젝트 디렉토리로 이동
cd aud-website

# 웹 서버 실행 (예시)
# Python을 사용하는 경우
python -m http.server 8000

# 또는 Node.js를 사용하는 경우
npx serve
```

브라우저에서 `http://localhost:8000`으로 접속

## 프로젝트 구조

```
aud-website/
├── index.html          # 메인 페이지
├── css/               # 스타일시트
├── js/                # JavaScript 파일
├── images/            # 이미지 리소스
└── README.md          # 프로젝트 문서
```
